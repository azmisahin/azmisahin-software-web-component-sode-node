# sode

sode ( sōd ) is browser node. The node converts the javascript file into a browser javascript file.

## Usage

> main.js

```js
/// Sode Reference
var Sode          = require('sode')

/// Sode
var sode          = new Sode()

console.log       `------------------------------------------------sode`
console.log       sode

```
